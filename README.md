## Overview

Synthetic Sentinel is an NLP pipeline that distinguishes **human-written text** from **AI-generated content** by combining:

- **RoBERTa** — fine-tuned transformer for contextual classification
- **Perplexity** — GPT-2 based text predictability score (AI = lower)
- **Burstiness** — sentence-length variance (humans = higher)

The system outputs a **Synthetic Probability Score** (0–100%), a binary label, and explainability visualizations.

## Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Run the data pipeline
python data_loader.py                    # Full pipeline (with perplexity)
python data_loader.py --skip-perplexity  # Faster, for quick testing

# 3. Train models
python train_model.py                     # Full training (baseline + RoBERTa)
python train_model.py --baseline-only     # Only Logistic Regression

# 4. Evaluate
python evaluate.py

# 5. Launch demo
streamlit run app.py
```
